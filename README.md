# MediTrack Pharmacy Stock Management System

This project is a full-stack DBMS assignment built as a **medicine and stock management system** for pharmacies or clinics. It includes:

- A responsive web front-end
- A Python backend with REST-style endpoints
- A relational SQLite database
- SQL scripts demonstrating schema design, constraints, indexes, views, triggers, and seeded sample data
- A report with the problem definition, ER design, normalization, and implementation details

## Run the project

```bash
python3 server.py
```

Then open `http://127.0.0.1:8000`.

## Project structure

- `server.py` - backend server and API routes
- `sql/schema.sql` - DDL, constraints, indexes, views, and triggers
- `sql/seed.sql` - sample DML data
- `static/` - front-end files
- `report/project_report.md` - project report
